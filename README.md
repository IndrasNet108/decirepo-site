# decirepo-site

Public website for `decirepo.com`.

This repository contains only the site layer (UI + public static API snapshots) and excludes private DLX core.

## Scope
- Pages and assets for public DeciRepo portal
- Public discovery metadata (`.well-known`)
- Public API snapshots for static rendering

## Non-goals
- No private DLX engine code
- No internal operational secrets
